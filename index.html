Rocket Simulation

This was my coursework for my C++ programming and design, and my real time graphics module in my 4th year at university. The aim of the project was to create a simulation of a rocket launch using the knowledge I had gained of various graphical techniques, the DirectX 11 pipeline, and C++.

Entity Component System(ECS) Game Engine

This was my coursework for my game architecture modile in my 3rd of university. The aim was to create an ECS game engine, and then use that engine to make a simple maze escape game. In the game the player must collect 3 keycards which can be found in the maze and then return to the escape portal.
